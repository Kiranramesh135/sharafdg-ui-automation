# Sharaf DG Automation Assignment

## Features

- Registration Flow
- Login Flow
- Product Purchase Flow (Add to Cart → Checkout → Payment)

## Tools Used

- Java
- Selenium WebDriver
- TestNG
- Maven
- Page Object Model (POM)

## Setup Instructions

1. Install Java 11+
2. Install Maven
3. Clone the repository
4. Run `mvn clean test`

## Test Execution

mvn test
